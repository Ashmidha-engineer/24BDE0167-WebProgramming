Online Theme Park Booking Website

Overview

This project is a web-based application developed to facilitate online booking for a theme park. It provides users with an intuitive interface to explore available rides, view pricing details, and access essential information such as contact details. The application is designed to deliver a smooth and responsive user experience across devices.

Key Features Home Page -A visually engaging landing page that introduces the theme park and provides navigation to all sections of the website.

Rides Section -Displays a collection of available rides with images and brief descriptions to help users explore attractions.

Pricing Information -Provides clear and structured ticket pricing details for different categories of visitors.

Contact Section -Enables users to view contact details or submit queries through a form interface.

Technologies Used Frontend: React.js, HTML, CSS, JavaScript Version Control: Git and GitHub
